## Asset Repository

All brand assets are available in the [public-awesome/assets](https://github.com/public-awesome/assets) GitHub repository.

## Usage Guidelines

- Use the gradient or pink logo on dark backgrounds
- Use the white logo on colored backgrounds
- Don't stretch, rotate, or alter the logo proportions
- Maintain clear space around the logo
- Don't add effects like shadows or outlines
